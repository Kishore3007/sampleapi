- Install Python 3.12.2 and make 
- then open this folder via Visual Studio Code
- Create a virtual env in your directory by using this command `` in your terminal 
- You will find a folder `venv` in your directory.
- Then type ` .\venv\Scripts\activate ` command in your terminal, to install libraries
- Then type this command ` pip install -r requirements.txt ` This will install all libraries necessary to execute the python code.
- Make sure you create .env file and 'OPENAI_API_KEY = <assign api key here>'
- just run the app.py file to locally host the application.
